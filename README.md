# vk-webhook
Сырая версия, PoC так сказать.

## Что это?

Это первоначальная (сырая) версия прослойки VK API, в частности, получения информации. Вывод информации идет в WebSocket.

## Как это работает?
Просто. В конфигурации указываем нужные настройки (CB/LP и их данные) и порт для веб-сокета. Затем он шлет полученные данные в этот вебсокет. Пример клиента описан в файле client.js.
